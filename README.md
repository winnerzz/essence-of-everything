# 万物

> 用第一性原理，重新理解每一个领域的核心脉络。

## 项目简介

**万物的本质**是一个跨学科的知识探索项目，通过 22 篇深度长文，以叙事化、第一性原理的方式，剖析数学、物理、统计、计算机科学、生命科学、社会与文明、认知科学、复杂系统等领域的根本逻辑。

每篇文章都以"一场……的推演"为视角，用隐喻、类比和思想实验替代教科书式的定义，帮助读者真正理解一个领域的本质——而不只是记住它的术语。

主页提供一个**交互式知识图谱**，通过力导向图可视化 22 个主题之间的关联关系，引导读者跨领域探索。

## 在线体验

直接在浏览器中打开 winnerzz.github.io/essence-of-everything，即可进入交互知识图谱，点击节点跳转到对应文章。
